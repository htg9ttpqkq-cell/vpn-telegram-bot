# **App Name**: Edelia VPN: Secure Connect

## Core Features:

- Hero Section Display: Render the prominent hero section with compelling headlines, calls to action ('Get Started', 'Open Telegram'), and a visually striking VPN UI illustration or dashboard mockup, emphasizing cinematic lighting and blue neon glow.
- Dynamic Stats Showcase: Implement an interactive statistics section featuring glowing cards for key benefits (e.g., 'Ultra Fast Servers') and animated counters for metrics like '99.9% uptime' and '<50ms latency'.
- Premium Features Presentation: Display a curated list of modern premium features (e.g., VLESS Reality, high-speed streaming) using elegantly designed cards that incorporate glowing borders and sophisticated hover effects.
- Guided Setup Flow: Present the 'How It Works' section as a clear, premium timeline or series of cards, guiding users through the 3-step setup process with appropriate icons.
- Interactive Dashboard Preview: Showcase a realistic, premium VPN dashboard UI featuring glassmorphism panels, demonstrating active subscription status, server location, traffic usage, and protected status.
- Flexible Pricing & Call-to-Action: Present transparent pricing plans (1, 3, 12 Months) in premium cards, visually highlighting the most popular option with a distinct blue glow, each with a 'Connect Now' button.
- Responsive and Adaptive Design: Ensure the entire landing page is built with a mobile-first approach, providing a seamless and highly polished experience across all device screen sizes, from mobile to desktop.

## Style Guidelines:

- Primary interactive and accent color: A vibrant electric blue, calculated as HSL(205, 85%, 55%), which translates to #2B9DEE, to signify speed, security, and digital prominence.
- Background color: An ultra-dark, almost black shade with a subtle blue tint, derived from HSL(205, 15%, 8%), resulting in #111517. This provides a deep, immersive dark canvas characteristic of modern cybersecurity interfaces.
- Accent lighting and glowing elements: A luminous cyan, approximately HSL(175, 70%, 65%), which renders as #67E4DA, used for neon glow effects, hover states, and to provide contrast and depth against the darker background.
- Headlines and prominent text: 'Space Grotesk' (sans-serif) for its futuristic, tech-oriented, and scientific feel, aligning with the modern SaaS aesthetic.
- Body text and functional information: 'Inter' (sans-serif) to complement 'Space Grotesk' with its neutral, machined, and highly readable modern aesthetic, ensuring clean typography throughout.
- Utilize 'Lucide Icons' with a modern, outline style. Icons should be styled with glowing blue/cyan effects to seamlessly integrate with the overall dark, futuristic UI and visual aesthetic.
- Implement 'Apple-level spacing' with generous use of whitespace, well-defined sections, and a structured grid system to ensure premium clarity and easy readability on all devices. Sections should feature 'cinematic lighting' and a 'modern SaaS layout' with elements like 'floating cards' and 'subtle gradients everywhere'.
- Incorporate 'ultra smooth animations' and 'modern hover effects' using Framer Motion. This includes subtle element entrances, smooth transitions for card interactions, animated counters, glowing border effects, and soft blur effects for depth and focus, contributing to a highly polished user experience.
